Changes I made:

General

I changed how the dropdown searchbar looked and placement. Now it fits better with the rest of the design. It also looks much better and is easier to use as it is larger now. I would also like to know how to do that when it is not searching for a specific product from an api but rather text in a html. How would one go about displaing the search results? Some general fixes as removing the margin on the body and fixing the label for the inputs that did not have id's.

I did not want it to do this as I wanted it to be as accessible as possible for people that use desktops, it also makes more sense with regards to interaction design.
